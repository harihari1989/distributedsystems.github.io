# Distributed Systems Visual Guide

A professional-grade GitHub Pages site for distributed systems patterns, interview prep, and real-world architecture reference.
Content lives in Markdown pages under `pages/` and uses Mermaid for diagrams.

## Structure
- `index.md` is the landing page.
- `pages/` contains topic pages.
- `assets/` holds CSS, JavaScript, and images.
- `scripts/` contains local datasets for interactive visuals.

Follow `AGENTS.md` for content and formatting rules.
